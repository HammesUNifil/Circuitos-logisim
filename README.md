## Half-subtractor
### Entradas: o circuito possui duas entradas (A,B)
### Saidas: o circuito possui duas saidas (Difference, Borrow)
### Funcionalidade: Utilizado para a subtração de dois bits

## 4-bit-Adder
### Entradas: o circuito duas entradas de 4 bits, o qual será somada posteriormente
### Saídas: a saida é a soma binaria dos valores de entrada
### Funcionalidade: O circuito soma valores binarios utilizando circutios de half-adder e full-adder
