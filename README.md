##DMUX (Demultiplexador) – 2 bits
Entradas: Uma unica entrada de dados e duas linhas de controle de dois bits
Saidas: Quatro possiveis saidas
FUncionalidade: O DMUX faz o contrario do MUX. Pegando um unico sinal de entrada e usando os 2 bits do selector para enviar esse sinal para uma das saidas

## MUX (Multiplexador)
Entradas: Possui várias entradas de dados (2, 4, 8, 16...).
Saídas: Uma única saída.
Funcionalidade: Recebe varias entradas diferentes, e e após isso, manda apenas o sinal escolhido pelo controlador para a saída.

## Full-subtractor
Entradas: A, B e Borrow In (normalmente são 3 entradas, não 2)
Saídas: Duas saidas (diff e borrow)
Funcionalidade: Realiza a subtração completa entre três bits

## Half-subtractor
Entradas: Duas entradas (A,B)
Saidas: Duas saidas (Difference, Borrow)
Funcionalidade: Utilizado para a subtração de dois bits

## 4-bit-Adder
Entradas: Duas entradas de 4 bits
Saídas: Uma saida sendo o resultado da soma binaria
Funcionalidade: O circuito soma valores binarios utilizando circutios de half-adder e full-adder
